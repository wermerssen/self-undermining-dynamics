# Self-Undermining Dynamics in Open Political Systems  
*A Structural Model*

This repository contains the official website and documentation for the research project **Self-Undermining Dynamics in Open Political Systems**, a structural model investigating how political systems generate, amplify, and eventually undermine their own stability through recursive dynamics.

The project provides a conceptual and structural framework for analyzing open political systems, focusing on:
- recursive feedback mechanisms  
- structural destabilization  
- self-undermining trajectories  
- meta-level transformations  
- long-term systemic evolution
